# Archive — old aircool.net.pl

This folder holds the previous version of the website that was live on cyberfolks.pl up until the rework in 2026.

## Contents

- `public_html_2022/` — the full deployed site (Bootstrap 3 + jQuery, Polish content, ~2022 vintage). This is what was on the FTP server before the new version went live.
- `original_index.html` — the original landing page HTML kept as a reference / rollback point.
- `sitemap.xml` — the original sitemap pointing at `index.php?p=N` URLs.

## Why keep this

If anything ever breaks with the new site and you need to roll back fast, you can re-upload `public_html_2022/` contents into the `public_html/` directory on cyberfolks via FTP. The site will be back to the old look in minutes.

## What replaced it

See `../new_website/` — the rebuilt version with modern design, multi-page SEO, schema.org markup, F-GAZ trust section, etc.
